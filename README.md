# Stock Predictions using Linear Regression and LSTM

This project compares linear regression and LSTM models for stock price prediction using financial time-series data. The analysis uses SPY data and evaluates different modeling approaches through rolling-window and expanding-window prediction frameworks.
## HTML Page
https://melker01olofsson-cyber.github.io/stock-predictions-lr-lstm/
## Methods

- Financial time-series analysis
- Feature engineering using returns and rolling volatility
- Linear regression benchmark models
- Rolling-window and expanding-window evaluation
- LSTM modeling for sequence-based prediction
- Model evaluation using RMSE and MAE

## Tools

- R
- Quarto
- quantmod
- dplyr
- ggplot2
- Keras
- reticulate

## Files

- `stock_predictions_lr_lstm.qmd` – source code and analysis
- `index.html` – rendered HTML report

## Key Takeaways

The project demonstrates how both traditional statistical models and neural networks can be applied to financial forecasting. The results should be interpreted cautiously, since stock price prediction is difficult and price-level forecasts are strongly influenced by autocorrelation.
